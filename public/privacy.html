Privacy Policy

Checkout Geo Flash for Ecwid

Effective date: 2026-04-06

Checkout Geo Flash is a static Ecwid app that lets merchants configure privacy-safe purchase notification behavior from the Ecwid admin dashboard.

Data We Process

  • Merchant configuration saved to Ecwid App Storage under the merchant's own store
  • Ecwid admin iframe payload details needed to authenticate store-level configuration writes
  • Optional support request details supplied directly by the merchant

Data We Do Not Use In This Static Deployment

  • No custom backend database
  • No Redis or long-running server-side session store
  • No cross-store analytics collector included in this repository

Merchant Control

Merchants control the public settings published by the app. The storefront runtime reads the merchant's public app configuration from Ecwid when enabled.

Privacy Requests

Contact the support channel listed in the live marketplace listing for privacy and data requests.
