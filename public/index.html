Checkout Geo Flash for Ecwid

This app is built for the Ecwid business owner inside the admin dashboard. It lets the merchant configure Checkout Geo Flash, preview the notification style safely, and publish privacy-safe settings without running a custom Node.js server, database, or cache.

⚡ Quick Start Guide — How This App Works Hide

Checkout Geo Flash adds social-proof toast notifications to your storefront. When shoppers browse your store, they see small pop-up messages like "Someone from Austin just paid for Weekend Tote" — creating urgency and trust. This dashboard is where you configure those messages.

  1. Configure your notifications below. Choose a runtime mode, write your sample messages, and set timing. These are the messages your shoppers will see.
  2. Use "Dashboard Preview" to test the look. The preview runs only in this dashboard and is never visible to shoppers. It lets you see how the toast will appear before going live.
  3. Save your settings. Click "Save To Ecwid" to publish your configuration. The storefront runtime reads these settings automatically.
  4. Your shoppers see real notifications on the storefront. Once saved and installed, every visitor to your store will see the toast notifications you configured. Check the Live Storefront Status panel below to confirm what's active.
Where is my real data? This app displays social-proof notifications — it is not an analytics tool. To see real order metrics, open Ecwid Admin → My Sales → Orders. To track store traffic and conversions, open Ecwid Admin → Analytics.

Setup Checklist

  5. Install the storefront runtime using the snippet below or automatic app injection.
  6. Visit your live storefront to confirm notifications are showing.
