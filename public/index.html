Merchant Setup

Steps the store owner or implementer follows from the dashboard.

  1. Connect this page inside the Ecwid admin so the app can load the store payload.
  2. Adjust the sample copy, location mode, and runtime behavior for this store.
  3. Save settings to Ecwid App Storage.
  4. Install the storefront runtime using the snippet below or automatic app injection.
